## Testing with JUnit 5 (Eclipse)

Do the following in Eclipse:

1. Right-click `RentalProject`
2. Click `Build Path`
3. Click `Add Libraries...`
4. Choose `JUnit`
5. Click `Next`
6. Select `JUnit 5`
7. Finish
8. Apply and Close
